# Security Policy

## Supported Versions

Generally, only the latest version of Germanized for WooCommerce has continued support. If a critical vulnerability is found in the current version of Germanized for WooCommerce, we may opt to backport any patches to previous versions.

## Reporting a Vulnerability

Germanized for WooCommerce is an open-source plugin for WordPress.
Please report security bugs found in the source code of the Germanized for WooCommerce plugin through the [Patchstack Vulnerability Disclosure Program](https://patchstack.com/database/vdp/4ebb0254-1bf0-407c-adf3-f510e6d03081). The Patchstack team will assist you with verification, CVE assignment, and notify the developers of this plugin.

Our most critical targets are:

Germanized for WooCommerce (this repository)
vendidero.de, vendidero.com -- the primary marketplace and marketing site, and all of it subdomains, e.g. download.vendidero.de
